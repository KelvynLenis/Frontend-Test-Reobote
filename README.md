# Teste Frontend - Kelvyn

### Faça uma copia local:

```
  git clone https://github.com/KelvynLenis/Frontend-Test-Reobote.git
```

### Instale as dependencias:

```
  npm install
```

### Execute o projeto:

```
  npm run dev
```

### O projeto estará disponível em:

```
  http://localhost:5173/
```
